Kenshin Impact

Kenshin Impact is an action role-playing game developed and published by Mihoyoman. The game features an open-world environment and action-based battle system using elemental magic and character-switching, and uses gacha game monetization for players to obtain new characters, weapons, and other resources. The game is online-only and features a limited multiplayer mode allowing up to four players to play together
